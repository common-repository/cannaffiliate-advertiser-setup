Set up CannAffiliate conversion tracking to allow influencers and publishers to start driving customers to your site.

== Description ==

Install this plugin to automatically set up post-back URL tracking to your website. All you need is your CannAffiliate Offer ID (provided by CannAffiliate support) in order to implement on your website.

This [CannAffiliate Advertiser](https://cannaffiliate.com/advertise/) plugin utilizes [TUNE API](https://developers.tune.com/) and requires agreement to both the [TUNE Terms of Use](https://www.tune.com/resources/data-and-privacy/terms-of-use/) and [CannAffiliate Terms & Conditions](https://login.cannaffiliate.com/terms) before use.

== Installation ==

How to install the plugin and get it working.

1. Upload the plugin file "cannaffiliate-advertiser-setup.php" or search "CannAffiliate Advertiser Setup" in the WordPress Plugin Directory

2. Activate the plugin 

3. Look for "CannAffiliate Setup" in the Tools section of your WP Dashboard

== Frequently Asked Questions ==

= Where do I find my Offer ID? =

Contact CannAffiliate support (support@cannaffiliate.com) to get your Offer ID.
